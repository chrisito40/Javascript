🛰Javascript

Javascript es un lenguaje de programación dinámico y de alto nivel para crear interfaces interactiva y dinámicas.

Es uno de los tres pilares fundamentales del desarrollo web.

  • HTML -> Define la estructura del contenido. (Elementos que aparecen en la página)
  • CSS -> Define la apariencia (colores,tamaños, estilos)
  • Javascript (JS) ->Define el comportamiento (Lo que hace la página o sus elementos.)

Características de Javascript

  1. Lenguaje interpretado -> No necesitamos compilarlo. El navegador lo entiende directamente.
  2. Basado en objectos -> Maneja objectos como document, window, arrays, date, class.
  3. Dinámico -> Podemos modificar el contenido en tiempo real. (Sin recargar)
  4. Versatil y de fácil aprendizaje -> Aunque nació para la web, se usa también en servidores, aplicaciones móviles, videojuegos y hoy en día en Inteligencia Artificial.
